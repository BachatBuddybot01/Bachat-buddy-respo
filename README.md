# 🛍️ Bachat Buddy Deal Bot

A simple Telegram bot that automatically fetches deals from popular deal channels and posts them into your own Telegram channel.

Built for beginners using **Python + Railway**.

---

## 🚀 Features

* 📡 Fetch deals from Telegram channels
* 🔗 Extract links automatically
* 📢 Auto-post to your Telegram channel
* ⚡ Lightweight & fast
* ☁️ Runs 24/7 on Railway

---

## 🧠 Tech Stack

* Python
* Telethon
* aiohttp
* Railway

## ▶️ Run Locally

Install dependencies:

```
pip install -r requirements.txt
```

Run the bot:

```
python main.py
```

---

## 🚀 Deploy on Railway

1. Push this repo to GitHub
2. Go to Railway
3. Click **New Project → Deploy from GitHub**
4. Select your repo
5. Add environment variables
6. Done ✅
